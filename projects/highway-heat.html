HighwayHeat

Project details page

Role: Systems Engineer

Summary

Highway Heat is an infinite vertical shooter inspired by the arcade classic 1942. Players are dropped into nonstop vehicular combat where the core challenge is to survive as long as possible. As each run continues, enemy intensity increases and players earn upgrades that improve firepower, speed, and survivability, creating a constant sense of progression and pushing players to see how far they can go before being overwhelmed.

What I Worked On

  • Built controller support using GateWare to ensure consistent input handling across gameplay and menus.
  • Integrated IMGUI and used it to implement/debug key UI systems, including an in-game health bar.
  • Designed and maintained standard enemy systems, including AI behavior, spawn logic, attack patterns, formations, and off-screen return handling.
  • Implemented image-loading support with stb_image.h for splash screen assets.
  • Created splash screen animation sequences and integrated them into startup flow.
  • Developed and managed the UI Manager, including pause menu and death menu flows.
  • Implemented pause menu actions (Resume, Main Menu, Quit) and death screen leaderboard display.
  • Built the AudioManager and added volume controls to the pause menu.
  • Improved startup and shutdown performance while addressing stability concerns.
  • Performed memory leak testing and fixed identified memory issues.
  • Used NDC-space bounds to implement invisible gameplay walls for both players and enemies.
