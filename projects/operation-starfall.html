Operation StarFall

Project details page

Role: Gameplay Engineer

Summary

Operation StarFall is a fast-paced sci-fi FPS set on an orbital station under attack. Players combine fluid traversal and combat to complete layered objectives, including surviving enemy waves, collecting keycard fragments, defending critical systems, and pushing toward a final control-room boss fight to prevent station destruction.

What I Worked On

  • Assisted with foundational player movement systems and general gameplay responsiveness.
  • Implemented a grapple hook system using gravity-based swinging mechanics.
  • Built inventory workflows, including base item definitions with ScriptableObjects.
  • Implemented item pickup and drop interactions tied to the inventory system.
  • Developed persistence for character, inventory, and weapon data across level transitions.
  • Implemented save/load systems using JSON files for game data persistence.
  • Managed web build setup and support tasks for playable deployment builds.
  • Handled general bug fixes across multiple gameplay and system areas.
